# MSBandEvaluation

iOS app to get sensing data from the Microsoft Band 2 and the Polar H7 chest strap. 

## Collected Data

Microsoft Band 2:
* RR intervals
* Heart rate
* Skin temperature
* Skin conductance (GSR)
* Acceleration
* Grysoscope data
* Altitude
* Ambient light level
* Ambient barometer level

Polar H7:
* RR intervals
* Heart rate
